# COMANDOS  
- git version ---(Verificar si Git está instalado.)  
- git config global user.name "Santiago Marin" ---(Configurar nombre de usuario.)  
- git config global user.email "santiago.marinb@upb.edu.co" ---(Configurar correo electrónico.)  
- cd ---(Cambiar de directorio.)  
- mkdir Repo1 ---(Crear carpeta llamada Repo1.)  
- cd Repo1 ---(Entrar a la carpeta Repo1.)  
- git init ---(Inicializar un repositorio Git en la carpeta actual.)  
- git status ---(Ver estado del repositorio (archivos rastreados/no rastreados)).  
- git add README.md ---(Añadir archivo específico al área de preparación.)  
- git add . ---(Añadir todos los archivos nuevos/modificados al área de preparación.)  
- git commit -m "Mensaje del commit" ---(Guardar cambios con un mensaje descriptivo.) 
- git log ---(Ver historial de commits en una sola línea por cada uno.)
- git remote -v ---(Ver a qué repo estás conectado)
- git remote set-url origin https://github... ---(Cambiar la URL del repo)
- git remote remove origin
git remote add origin https://github.com/USUARIO/NUEVO-REPO.git ---(Quitar el repo y poner otro nuevo)
- git clone https://github.com/USUARIO/REPO.git ---(Clonar otro repositorio)
